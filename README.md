# NalTool

A lightweight cross-platform encryption tool.  

---

## English

### Description

NalTool is a lightweight cross-platform encryption tool for text and files.  
NalTool features a command-line interactive interface for a user-friendly experience.  
It follows the principles of simplicity and compatibility.  
Pure Python implementation. Requires Python 3.6 or higher. No external dependencies.  

### Features

- Encrypt and decrypt text  
- Encrypt and decrypt files  
- Generate NalKey key files for easy key management  
- Cross-platform: Windows, Linux, macOS  

### Language

This repository contains two versions of NalTool:  
- NalTool-version-English.py      - English version  
- NalTool-version-Chinese.py      - Chinese version

Both versions are identical in functionality, only the interface language differs.  

### NalKey

NalKey saves your key to a file for automatic loading.  
Warning: NalKey uses obfuscation, not encryption. Keep your key files safe.  

### NalFile

NalFile is the encrypted file format generated by NalTool, compatible across multiple platforms.  
File structure: NALT Magic(4) + Version(1) + Salt(16) + IV(16) + HMAC(32) + Chunked Encrypted Data(Variable)  

### Quick Start

git clone https://github.com/TasKin-tk/NalTool.git  
cd NalTool  
python NalTool-version-language.py  

Replace version and language with the actual version number and language

## 中文

### 简介

NalTool 是一款轻量级跨平台加解密工具，支持文本和文件加密。  
NalTool 基于命令行交互界面，具有友好的使用体验。  
其秉承着简洁与兼容的理念。  
纯 Python 实现，仅需 Python 3.6 或更高版本，无任何外部依赖。  

### 功能

- 加密和解密文本  
- 加密和解密文件  
- 生成 NalKey 密钥文件，方便密钥管理  
- 跨平台支持：Windows、Linux、macOS  

### 语言

本仓库包含两个版本的 NalTool：  
- NalTool-version-English.py      - 英文版  
- NalTool-version-Chinese.py      - 中文版

两个版本功能完全相同，仅界面语言不同。  

### NalKey 说明

NalKey 可将密钥保存为文件，使用时自动加载。  
注意：NalKey 使用混淆存储，并非加密。请妥善保管密钥文件。  

### NalFile 说明

NalFile 是 NalTool 生成的加密文件格式，兼容于多平台。  
文件结构：NALT 魔数(4) + 版本(1) + Salt(16) + IV(16) + HMAC(32) + 分块加密数据(可变)  

### 快速开始

git clone https://github.com/TasKin-tk/NalTool.git  
cd NalTool  
python NalTool-version-language.py  

将 version 和 language 替换为实际版本号和语言。
